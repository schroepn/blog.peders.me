Weekly Learns

This week I didn't really do much research on HTML or CSS or anything really because im still getting into the groove for school. You know how it is, you dont do anything an entire summer and when you have to start doing things again you forget how to do anything. I guess I better look for something interesting to learn now.

One thing I found out: how to make a linebreak. As I was writing the above I realized I didn't know how to add a linebreak to html, I figured it out from here[1]. On that same page I learned that I shouldn't use them in case someone with a screen reader reads my page, instead I should use "<p>"".

The next thing I learned was how to add the <"things within angle brackets">. I checked the mdn html homepage[2] and while it didn't show how to do it, it had multiple examples of using it by putting the brackets within <code> and adding quotation marks. I then looked around for how to actually put brackets (and other special character) into html, and I found a website here[3] which taught me to use &lt; for < and &gt; for >. After another google search I found a good reference tool [4] with a bunch of html character reference codes that I could use.
