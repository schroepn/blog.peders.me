Weekly Learns

This week I decided to try to make the blog look a bit better/more aesthetically pleasing. As you can see by comparing this blog post to last weeks, there has been improvement. Albeit maybe not that much. I used the examples in the reading for the class [1] to create the monocolumn. But I wanted to have a header that popped out. So after taking the reading and adding some alignment[2] and color changing, I got a new fancy header that says the week for the blog Post.

At some point I want to get a following header, that follows the web page down. Like what happens to the google search bar[3] when you scroll down. I guessed that it was from the style="position: fixed; top: 0px;" And after playing around with it a bit, I figured out it was from position:fixed. So I added it to this page. Which had… not so great of an outcome.

But it was staying at the top of the page when I scrolled down, so there's that. I just needed to recenter it. Which was pretty easy, all I did was make it span the entire page instead of just try to get it to cover only the middle 50%, then I just center aligned the text using text-align: center;. The final problem: get the main content to sit comfortablly under the header. I thought it would be just as easy as adding a margin to the main, but it also adeded some margin to the header. So I added padding instead, so now there is technically some empty space under the header, but that shouldn't really matter, right?
