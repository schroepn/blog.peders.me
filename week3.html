Weekly Learns

This week, I wanted to improve the look of the blog even more, so time for sidebars with links! Looking at MDN's website[1], it seems to use an aside for the sidebar. So, now time to add an aside!

Yeah Yeah, formatting with CSS. I am going to make an educated guess that they used a flexbox. And if we check it… I'm not sure. The CSS I can see talks about grid-row and grid-column. Which from what I can understand from MDN[2] means that the page is layed out like a 2d grid (whodathunk). According to HubSpot[3] the difference between flex and grid is that flex is 1d, while grid is 2d.

After looking back at the mdn screenshot, I noticed that everything is put into a root div. Which probably makes it easier when using both. Although, I guess I would change whether or not the header was in the root depending on if i used flex or grid. I am going to try to use flex, because grid includes grids and, as we all know, grids are scary. Now I should probably learn about color theory and learn what colors look good with each other… Nah.
